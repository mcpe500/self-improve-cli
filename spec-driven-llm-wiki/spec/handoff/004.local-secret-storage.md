# Handoff: Local Secret Storage

## Session Summary

Implemented local provider API key storage so users do not need environment variables. Provider keys are stored in `.selfimprove/secrets.json`, config stays non-secret, `/connect` prompts for key, `/key` replaces it, and provider calls prefer stored secret over env fallback.
